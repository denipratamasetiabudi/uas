Tips Mengatasi Laptop atau Desktop Lemot

pada masa pandemi covid 19 membuat kita harus bekerja di rumah atau dikenal dalam istilah work from home. tentunya kita memerlukan pc/laptop untuk bekerja. namun, terkadang ada beberapa kendala salah satunya adalah laptop yang lemot. tapi, jangan panik ada beberapa cara yang bisa kita lakukan untuk mengatasi laptop yang lemot tersebut yang pasti caranya sangat sederhana dan mudah diikuti.

kalian juga perlu tahu kemungkinan laptop lemot disebabkan karena laptop terlalu banyak dalam menjalankan program. hal itulah yang mempengaruhi kinerja dan kecepatan laptop tersebut.

Akan tetapi tidak perlu khawatir, banyak juga tips mengatasi laptop lemot agar hal tersebut tidak terulang lagi dan menganggu aktivitas pekerjaan kalian. Berikut adalah cara mengatasi dan tips mengatasi laptop lemot

*Mengatasi Laptop Lemot pada Dekstop Windows

-Identifikasi Program yang Membuat Laptop Lemot Jika kalian memiliki program yang menggunakan banyak memori, maka ini saat uang tepat untuk memutuskan apakah program tersebut sepadan dengan sumber daya yang digunakan. Atau bisa juga program yang dibuka sebelumnya dan telah kita lupakan. Menghapus program ini dapat menjadi salah satu cara tercepat mengatasi laptop lemot.

-Tutup Program yang Tidak Perlu Untuk mempercepat komputer yang lambat, tutup program yang tidak perlu. Untuk melakukannya, tekan Ctrl, Alt, dan Delete. Pilih Task Manager dan pada Processes tab, kita dapat melihat apa yang terbuka dan berapa banyak CPU dan memori yang digunakannya. Setelah itu, tutup program yang tidak digunakan.

Komputer sering kali memiliki program yang diaktifkan saat kita menyalakan komputer. Atau, ada program yang diperlukan pada awal start-up, lalu tidak diperlukan lagi. Menonaktifkan ini dapat menjadi solusi mengatasi laptop lemot.

-Periksa Browser Web dan Koneksi Intermet Jika kalian melihat laptop mulai lemot, pastikan untuk memeriksa browser yang digunakan apakah up to date atau tidak. Jika ada opsi untuk menginstal versi baru secara otomatis, aktifkan. Memiliki terlalu banyak Add-On dan Extensions juga dapat memperlambat browser di laptop.

-Defrag Hard Disk Drive Defragmenting atau defragging hard disk drive (HDD) dapat mengurangi waktu yang dibutuhkan sistem untuk menemukan setiap bagian data yang dibutuhkan untuk menjalankan program dan aplikasi. Mengatasi laptop lemot dengan mendefrag disk dapat dipakukan dengan klik menu Start atau tombol Windows, pilih Control Panel, lalu System and Security. Di bawah Administrative Tools, klik Defragment your Hard Drive.

Terdapat beberapa cara mengatasi laptop lemot agar hal tersebut tidak terulang lagi.
Misalnya, mengurangi atau uninstall penggunaan persangkat lunak yang tidak perlu,
agar kinerja sumber daya sistem tetap terjaga. Terdapat beberapa tips mengatasi laptop lemot
lainnya, yaitu:

1. Hentikan Program yang Berjalan Otomatis Menghentikan program yang berjalan otomatis saat melakukan startup dapat membuat laptop lemot. Misalnya adalah aplikasi antivirus. Stop penggunaan otomatis dan gunakan saat memang benar-benar membutuhkan program tersebut.

2. Update Windows, Driver, dan Aplikasi Banyak yang bilang bahwa mengupdate perangkat lunak adalah ide yang bagus untuk keamanan. Ini benar dan juga dapat menjadi salah satu tips mengatasi laptop lemot. Sistem operasi Windows akan secara otomatis mengirimi pemberitahuan ketika update tersedia. Pastikan kalian tidak terus menundanya. Jika melewatkannya secara tidak sengaja, maka menu Start - Setting - Security Update - Update Windows.

3. Hapus File yang Tidak Perlu Data laptop yang terlalu penuh dan berantakan, rupanya dapat mempengaruhi kinerja. Terutama, jika file di dalamnya adalah file-file berukuran besar seperti gambar beresolusi tinggi, file audio, dan video.

Mengosongkan ruang dengan membiasakan menghapus file serta folder setiap minggu dan mengosongkan recycle bin sesudahnya dapat menjadi salah satu tips mengatasi laptop lemot.

Ini juga membuat kalian ingat dengan apa yang ada di file dan folder, sehingga tidak akan terlalu khawatir tentang kesalahan menghapus sesuatu yang penting.

4. Temukan File yang Menggunakan Sistem Besar Jika kalian merasa laptop kalian tiba-tiba lemot secara signifikan, kemungkinan ada program tertentu yang harus disalahkan. Salah satu cara untuk mengidentifikasinya adalah dengan masuk ke Task Manager dan mencari tahu apa yang menghabiskan sumber daya laptop kalian.

Klik kanan taskbar dan pilih Task Manager. Klik More details untuk mengetahui informasi tentang program yang sedang berjalan di laptop dan ini akan menunjukkan Processes Tab secara default.

Itulah beberapa cara mengatasi laptop lemot yang bisa dicoba sendiri di rumah. Jangan khawatir karena banyak cara yang bisa dilakukan untuk mengatasi laptop lemot tersebut.
